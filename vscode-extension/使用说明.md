# Inline Notes 插件使用说明

## 安装方式

### 方法 1：从 .vsix 文件安装（推荐）

1. **打开 VS Code**
2. **打开扩展面板**：
   - 按 `Ctrl+Shift+X` (Windows/Linux) 或 `Cmd+Shift+X` (Mac)
   - 或点击左侧边栏的扩展图标
3. **安装插件**：
   - 点击扩展面板右上角的 `...` 菜单
   - 选择 "从 VSIX 安装..." (Install from VSIX...)
   - 选择项目目录中的 `inline-notes-0.1.0.vsix` 文件
4. **重启 VS Code**（如果需要）

### 方法 2：开发模式运行（用于调试和开发）

1. **在 VS Code 中打开项目**
2. **按 `F5`** 或点击 "运行和调试"
3. **会打开一个新的 VS Code 窗口**（Extension Development Host）
4. **在新窗口中打开 Python 文件**，即可使用插件

## 使用方法

1. **打开任意 Python 文件**（`.py`）
2. **将光标悬停在代码上**，会显示：
   - **行解释**：说明这一行在做什么
   - **符号解释**：解释该行中的符号含义
3. **点击"展开更多"**：查看更详细的说明、常见误解和对比示例

## 示例

打开 `test.py` 文件，尝试悬停在以下代码上：

```python
x = 5          # 悬停查看 = 的解释
if x == 5:     # 悬停查看 == 的解释和行解释
    print("hi") # 悬停查看 print 和 () 的解释
```

## 卸载

1. 打开扩展面板 (`Ctrl+Shift+X`)
2. 搜索 "Inline Notes"
3. 点击卸载按钮

## 重新打包（修改代码后）

如果修改了代码，需要重新打包：

```bash
npm run compile
npx @vscode/vsce package
```

会生成新的 `inline-notes-0.1.0.vsix` 文件。
